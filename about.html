This Process Mining Breach Analyzer is a thesis project for Anish Automobiles. It demonstrates monitoring the standard workflow from RFQ to Payment Clearance, analyzing uploaded CSV logs to detect process breaches such as missing or out-of-order steps.

Built with Flask for backend analysis and a simple, responsive front-end for user interaction.
